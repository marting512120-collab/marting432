# RankFlow SEO — Agency Website

Professional, multi-page HTML/CSS agency website for RankFlow SEO, an SEO agency that helps local businesses and e-commerce stores get found on Google.

**Tagline:** Higher Rankings. More Traffic. Measurable Leads.

## Structure

```
index.html                   # Homepage — Hero, Problem, Services overview, Industries, CTA
services.html                # Services — Local SEO, E-commerce SEO, Audits, Content Marketing, Pricing
about.html                   # About — Mission, values, team, core principles
case-studies.html            # Case studies — 3 real-world examples with results
css/style.css                # Shared stylesheet (all pages)
blog/
  seo-cost-2025.html         # Blog: "How Much Does SEO Cost in 2025?"
  seo-vs-google-ads.html     # Blog: "SEO vs Google Ads: Which Builds a Better Business?"
README.md
```

## SEO Optimization

- **Schema markup:** JSON-LD LocalBusiness + Service on every page
- **Title tags:** Page-specific, targeting priority keywords
  - Home: "SEO for Small Business & E-commerce"
  - Services: "Local SEO Services | E-commerce SEO"
  - About: "About RankFlow SEO | We Grow Businesses"
  - Case Studies: "SEO Case Studies | Real Results"
- **Meta descriptions:** Include relevant keywords for each page
- **Semantic HTML5:** Proper heading hierarchy (h1-h4)
- **Target keywords woven throughout:** local seo services, ecommerce seo services, seo agency pricing, affordable seo services, technical seo audit, lawyer seo, dentist seo, plumber seo, shopify seo services
- **Blog posts target:** "how much does seo cost" (KD 36) and "seo vs google ads" (KD 48)
- **Footer:** Cross-linked services, industries, and company pages

## Design

- Pure HTML/CSS/JS — no frameworks or build tools
- Mobile-responsive (320px to 1200px+)
- Consistent navigation with active page highlighting
- Blue primary color (#2563eb) as specified
- CSS custom properties for theming
- Fast-loading (minimal HTTP requests, lightweight assets)
