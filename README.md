# RankFlow SEO — Agency Website

Professional, mobile-responsive HTML/CSS agency website for RankFlow SEO, an SEO agency that helps local businesses and e-commerce stores get found on Google.

## Structure

```
index.html                  # Main single-page website (home, services, about, case studies)
blog/
  seo-cost-2025.html        # Blog: "How Much Does SEO Cost in 2025?"
  seo-vs-google-ads.html    # Blog: "SEO vs Google Ads: Which Builds a Better Business?"
README.md
```

## Pages/Sections (all on one page)

| Section | Content |
|---------|---------|
| **Hero** | "Get Found. Get Leads. Grow Predictably." with dual CTAs |
| **Trust Bar** | Social proof with target industry badges |
| **Problem** | "Tired of Being Invisible on Google?" — Problem/Solution framing |

## SEO Optimization

- **Title tag:** Targets "SEO for Small Business & E-commerce"
- **Meta description:** Includes local SEO, e-commerce SEO, pricing keywords
- **JSON-LD Schema:** LocalBusiness + Service schema markup
- **Semantic HTML5:** Proper heading hierarchy (h1-h4)
- **Target keywords woven throughout:** local seo services, ecommerce seo, seo agency pricing, affordable seo, lawyer seo, dentist seo, plumber seo, shopify seo services
- **Blog posts target:** "how much does seo cost" (KD 36) and "seo vs google ads" (KD 48)

## Design

- Pure HTML/CSS/JS — no frameworks, no build tools
- Mobile-responsive (tested down to 320px)
- CSS custom properties for consistent theming
- Smooth scroll navigation
- Mobile hamburger menu
- Fast-loading single-file architecture
